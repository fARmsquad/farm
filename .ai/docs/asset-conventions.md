# Asset Conventions — FarmSim VR

## Naming Convention
```
[Category]_[Name]_[Variant]_[Size].[ext]

Categories:
  Crop_    — farming crops and plants
  Tool_    — farming tools (hoe, watering can, etc.)
  Env_     — environment props (fences, buildings)
  TX_      — textures
  MAT_     — materials
  SFX_     — sound effects
  MUS_     — music
  UI_      — user interface elements
  VFX_     — visual effects / particles
```
